Simple Integration

One Request. Decision-Ready Response.
Governance Included.

The API is purpose-built for high-reliability products: deterministic output contracts, confidence surfaces, and evidence metadata that can be rendered directly in your UX.

Standardized fields (answer, confidence, source surfaces, uncertainty flags) make it straightforward to implement confidence-aware workflows, escalation logic, and guardrailed automation.

// Request
POST /v1/query

{
  "query": "What is the current FDA approval status for pembrolizumab in NSCLC?",
  "domain": "medicine",
  "confidence_threshold": 70
}

// Response
{
  "answer": "Pembrolizumab is FDA-approved for first-line NSCLC with PD-L1 ≥1% (monotherapy) and in combination with chemotherapy regardless of PD-L1 expression...",
  "confidence": 94,
  "sources": 18,
  "verified": true,
  "uncertainty_flag": false
}
